# 条件渲染 Condition

## 结构
| 字段       | 必须  | 标题  | 类型                                | 默认值 | 说明  |
|----------|-----|-----|-----------------------------------|-----|-----|
| key      | 是   | 字段名 | String                            | 无   |     |
| operator | 是   | 操作  | String<[operator选项](#operator选项)> | eq  |     |
| value    | 是   | 值   | Any                               | 无   |     |

### operator选项

- eq：字段与值相等时渲染
